# AI-Based Blueprint Analysis for Intelligent House Color Visualization

## Overview

This project presents an AI-powered system that analyzes architectural blueprints and provides intelligent house color visualization. It automates manual blueprint interpretation using computer vision and machine learning techniques.

## Features

* Blueprint analysis using image processing
* Room and area detection using OpenCV
* OCR-based text and label extraction
* AI-based color palette recommendation
* Interactive 2D and 3D visualization
* Real-time color preview for interior design

## Technologies Used

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js
* AI/ML: Python
* Libraries: OpenCV, OCR
* 3D Visualization: Three.js

## Installation and Setup

### Clone the repository


git clone https://github.com/PreethiMendon/AI-Based-Blueprint-Analysis-for-Intelligent-House-Color-Visualization.git
cd AI-Based-Blueprint-Analysis-for-Intelligent-House-Color-Visualization


### Backend Setup


cd backend
npm install
npm start


### Frontend Setup


cd frontend
npm install
npm start


### Python Setup

cd python
pip install -r requirements.txt
python main.py
